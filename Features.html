Serverless Compute

Serverless computing dynamically manages the compute and network resources that your application requires. Instead of you manually provisioning the type and amount of memory, cores, and I/O, DCP matches your workload with the right infrastructure. .

API Centric Framework

DCP provides a unified API framework for your code to run anywhere, built from first principles to be intuitive for developers of all backgrounds. This API can be called anywhere from anywhere, from a CL to a website.

Run Serverless Anywhere

DCP converts owned hardware into a serverless computing cluster. Every device added to this pool of resources is one less that you have to manually configure. Bring the cutting edge of serverless to your own devices.

Zero Lock-In

Unlike other serverless computing solutions, DCP is entirely agnostic to the source of hardware. No matter how advanced your application, there is no requirement to migrate data or build for a specific cloud platform.

Performance Management

Dynamically adjust your application as it is executed. Useful for scaling up or down without having to restart work that is in progress, you can also get fine-grained runtime metrics to judge its performance.

Fast Cold Starts

Hardware is always ready to execute your functions, making the application begin almost instantly. Despite this, there is no performance loss when not in use. For users who have experienced serverless computing before, DCP solves the "cold start" problem.

Smart Data Parallelism

Simplify your code with compute.for( ), the function that automatically spreads data and instructions across any architecture. By coordinating the flow of parallel work, DCP is a point and click solution.

Application Ecosystem

Platforms have revolutionized the world, and many of the most popular software tools let users build and share apps. DCP fully supports lets users build, share, and utilize third party apps. From disease modelling to ML.

Engaging User Interface

DCP apps can take on almost every frontend imaginable, from beautiful HTML5 pages to other software packages. Your application users can now trigger advanced computations without writing a single line of code.

Interactive Computing

Build parameters into your apps that can be adapted for different experiments and datasets. DCP apps also allow others to back-check computational results for open science and research review.

Seamless Backend

Every app integrates with the DCP compute platform. Specify where your app user's workloads execute, whether on a local cluster or in the cloud. DCP also contains a deterministic math engine so results stay consistent, regardless of OS.

Potential to Monetize

Apps can be free to use, but they can also be licensed or paid for based on compute consumption. By building apps in an area of interest, developers can seamlessly monetize their work and earn compute themselves.

Largest Dev Community

DCP's execution engine is built on JavaScript, with native support for NPM. With over 1m packages, the largest software registry in the world is ready to be used in your app.

Complete Distributed OS

Distributed computing must improve the performance of jobs while minimizing overhead. This is difficult to achieve, but DCP solves the tough problems to make any architecture greater than the sum of its parts.

Optimized Scheduling

DCP's algorithms automatically optimizes the time- and space-sharing allocations based on the type of connected resources and user defined parameters. This dynamically updates as jobs are added or change nature.

Confirms Work Profiles

Easily estimate the size and compute requirements of your workload to accurately estimate its profile. DCP also cross-validates results that different nodes return within your project's confidence requirements.

Arbitrary Compute Grids

Build and manage your own personal compute cluster with spare servers and IoT devices. Share access to anybody, anywhere, and allocate specific nodes to different tasks. Grids can also be sub-divided into smaller and hyper-local networks.

Highly Elastic

Scavenge compute and memory from any machine based on consumption, time of day, type of application, and more. When resources are required locally, DCP is perfectly elastic and instantly scales back.

Manages Network Traffic

Automatically take different networks and communication protocols into consideration. From 5G and SSH to Ethernet and open WiFi connections, DCP's overlay network accounts for differences in bandwidth and latency.
